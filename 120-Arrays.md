# Function Checkpoint

Create a function that takes in an array. Add up first and fourth values in the array.

## Examples:

### Example 1

```
console.log(yourFunction([1, 2, 3, 4, 5]));
```

should log 

```
5
```

### Example 2

```
console.log(yourFunction([6, 7, 8, 9, 10, 11, 12]));
```

should log


```
15
```
